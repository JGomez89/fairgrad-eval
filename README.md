# fairgrad-eval
Implementation and empirical evaluation of the FairGrad in-processing algorithm for improving model fairness, based on Wang and Yang (arXiv:2504.14388). This repository reproduces the method, benchmarks its performance, and analyzes its impact on fairness–accuracy tradeoffs.
